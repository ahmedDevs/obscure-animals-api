Obscure Animal Species API

Types of requests our API handles

  • url/api
  • url/name species (ex: url/spectral bat)

Some of the species included in the API

  • spectral bat
  • velvet worm
  • vampire squid
  • tripod fish
  • penis snake
  • naked mole rat

Here's what the Objects contain

Every object is a species and contains the following:

  • scientific name
  • description
  • geographical origin
  • image
  • copyright
